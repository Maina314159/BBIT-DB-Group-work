Project Title -> WATCH SHOP MANAGEMENT DATABASE SYSTEM

PROJECT PURPOSE AND SUMMARY -> :
Create a fully functional and applicable Dataase schema derived from an ERD whilst applyng topics learned such as Normalization, Dependancies etc.
The proposed system is a Watch Retail Shop Database System designed to store and manage information related to watches,
suppliers, customers, employees, orders, payments, and warranties

By implementing this system, the shop will improve efficiency, reduce data redundancy,
maintain accurate sales records, and generate useful reports for business management.
